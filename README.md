# SINUX_V3
sinux v3 is open hardware evalution board for Allwinner V3s chip

### supported peripherals : 
* I2C
* SPI
* SD Card
* UART
* SPI Flash
* lradc
* LCD (40pin fpc)
* USB
* Backlight
* Audio
* RTC
* LAN
* GPIO

### Other features
* Resistive touch (NS2009 chip)
* Capacitive touch connector
